Η βασικη αρχιτεκτονικη εχει υλοποιηθει, οι κυριες κλασεις υπαρχουν και η προσομοιωση λειτουργει σε μεγαλο βαθμο.

## Περιγραφη Υλοποιησης

Η εργασια υλοποιει μια απλη προσομοιωση αυτοκινουμενου οχηματος σε ενα 2d array (grid world), με χρηση αντικειμενοστραφους προσεγγισης.

### Κοσμος
	•	Ο κοσμος υλοποιειται ως δισδιαστατος πινακας δεικτων σε αντικειμενα.
	•	Τα ορια του κοσμου καλυπτονται απο τοιχους.
	•	Καθε κελι μπορει να ειναι κενο η να περιεχει αντικειμενο.

### Αντικειμενα
Υπαρχει βασικη κλαση object και ιεραρχια με:
	•	κινουμενα αντικειμενα (car, bike)
	•	στατικα αντικειμενα (parked_car, traffic_sign, trafic_light)
	•	αυτοκινουμενο οχημα (self_driving_car)

Η διακριση μεταξυ κινουμενων και μη γινεται με virtual συναρτηση is_moving().

## Ροη Προσομοιωσης
	1.	Διαβαζονται οι παραμετροι απο τη γραμμη εντολων
	2.	Δημιουργειται ο κοσμος
	3.	Τοποθετουνται τα στατικα και κινουμενα αντικειμενα
	4.	Τοποθετειται το self driving car
	5.	Σε καθε tick:
	•	κινουνται τα κινουμενα αντικειμενα
	•	ενημερωνονται τα φαναρια
	•	γινεται αναγνωση δεδομενων απο τους αισθητηρες
	6.	Η προσομοιωση τερματιζει μετα απο συγκεκριμενο αριθμο ticks

## Προβληματα κατα την δημιουργια

Με προβληματισε πολυ η εννοια του direction, στην αρχη εκανα μια υλοποιηση με strings,επειτα ειδα οτι δεν ηταν καθολου αποτελεσματικος ο ελεγχος σε καθε loop με if(direction == "NORTH"),και εψαξα στο διαδικτυο και εκανα συνομιλιες με το gpt για να με βοηθησει για την υλοποιηση του.
